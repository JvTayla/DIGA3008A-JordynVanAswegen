Week 6 - Current Style Guide

By Jordyn Van Aswegen

Posted: March 24, 2025

Iteration 1

Overview:

The style guide for my portfolio website is inspired by a clean, comic book aesthetic, designed to be both visually striking and user-friendly. My goal is to create a site that reflects my artistic passion while also ensuring that the design is intuitive, easy to navigate, and professional. I want the site to have bold, dynamic colors, smooth rounded corners, and a minimalistic yet engaging layout that pulls from comic book influences without overwhelming the user.

Typography:

  • Primary Font - 'Roboto'

    I’ve chosen Roboto for most of the body text and navigation. It's a clean, versatile sans-serif font that’s easy to read on both mobile and desktop platforms. It complements the modern, minimalist style I’m aiming for and ensures that the website remains readable and functional.

    View Roboto

  • Secondary Font - 'Bangers'

    To bring in the comic book vibe, Bangers will be used for headings, titles, and impactful statements. It has a playful, bold style reminiscent of comic book lettering, which helps convey the energetic feel I'm after. It’s a great contrast to Roboto’s simplicity and adds a fun element to the design.

    View Bangers

  • Fallback Font - 'Arial'

    As a backup, I’m using Arial to ensure that the website maintains its readability and appearance across different devices and browsers. It's a safe, widely supported font that works well with both Roboto and Bangers.

    View Arial

Color Palette:

Primary Colors:

  • Deep Red (#A81000): This bold red serves as a focal point, drawing attention to key elements like headings, buttons, and calls to action. It’s vibrant and energetic, giving the site a dynamic feel without being overpowering.
  • Electric Blue (#3498DB): A bright, contrasting blue used for interactive elements such as hyperlinks, buttons, and hover effects, making the website feel more engaging.

Secondary Colors:

  • Bright Yellow (#F1C40F): A fun, eye-catching color used sparingly to highlight important details like icons, tags, or links. It adds an extra pop to the design.
  • Slate Gray (#292f34): Used for background elements and text, this neutral tone helps balance out the vibrancy of other colors and increases readability.

Neutral Colors:

White (#FFFFFF): Clean and simple, used for the main background to keep the site feeling open and airy.

Logo and Personal Branding:

For my personal logo, I’ve created a digital self-portrait that will be prominently displayed on the website. This stylized digital image represents me and serves as both a personal and professional brand mark. It helps create a unique identity and will be placed in the header for visual consistency.

Layout and Design Elements:

  • Rounded Corners:

    Used on buttons, containers, and cards to create a soft, modern aesthetic that complements the comic-inspired layout.

  • Whitespace and Hierarchy:

    Prioritizing whitespace avoids clutter and helps guide the user’s attention using visual hierarchy through font size, color, and placement.

  • Mobile Optimization:

    The mobile layout mirrors the color palette and typography but simplifies navigation using a collapsible menu and larger buttons.

  • Interactive Elements:

    Comic-inspired pop-up effects and hover transitions will make the browsing experience more dynamic and immersive.

>Future Developments with CSS:

As I move forward with CSS development, I plan to refine spacing (padding, margins), implement responsive design rules, and experiment with comic-inspired CSS animations. These may include hover transitions and comic panel-style scroll effects.

I want to strike a balance between personality and performance—ensuring the site remains user-friendly, polished, and expressive. This style guide sets the tone for how I’ll evolve my website as both a personal and professional space.
